# 🌟 芝秀月嫂官網 - 簡易內容管理系統

一個無需編碼知識就能修改網站內容的系統！

## 🚀 三步快速開始

### 1️⃣ 打開管理面板
點擊下方連結或在瀏覽器輸入：
```
admin.html
```

### 2️⃣ 快速編輯
- **服務項目**：修改標題、圖示、項目描述
- **客戶見證**：編輯評語、客戶名稱、頭像
- **首屏文案**：更新品牌訊息和描述
- **檔期狀態**：更新預約狀態和名額

### 3️⃣ 保存更改
點擊 **「保存所有更改」** 按鈕，改動會自動出現在官網上！

## 🎯 常用操作

### 添加新的服務項目
1. 進入「服務項目」標籤
2. 點擊「新增服務項目」按鈕
3. 填入：標題、圖示代碼、項目描述
4. 保存更改

### 添加客戶見證
1. 進入「客戶見證」標籤
2. 點擊「新增客戶見證」按鈕
3. 填入：評語、客戶名稱、圖片 URL
4. 保存更改

### 更新檔期狀態
1. 進入「檔期狀態」標籤
2. 修改檔期時間和剩餘名額
3. 保存更改
